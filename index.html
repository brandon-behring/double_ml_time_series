Double ML Time Series#

Book-grade companion documentation for the Double Machine Learning manuscript. This is not a production deployment package and the API is not yet public-stable.

The current code supports cross-sectional partially linear DML, temporal PLR DML with lagged controls, time-series cross-validation helpers, HAC inference, and synthetic companion examples. Deferred topics such as true dynamic g-estimation, period-specific treatment effects, causal forests, BLP/policy trees, and automatic stationarity enforcement are documented as future work unless an executable example or tested API proves otherwise.

Tip

Current status lives in docs/CURRENT_STATUS.md. Superseded roadmaps and reports are archived under docs/archive/superseded_2026-05-02/.
