# Real-world replications and applied DML

Real-world replications and applied DML — the user's second emphasized sub-area. Covers the 401(k) eligibility / participation literature (Poterba–Venti–Wise; Chernozhukov–Hansen; Abadie; Engen–Gale–Scholz; Benjamin), LaLonde / Dehejia–Wahba program-evaluation reanalysis (Smith–Todd; Diamond–Sekhon; Imbens), narrative macro-shock identification (Mertens–Ravn; Romer–Romer; Cloyne), Stock–Watson SVAR-IV applications, real-time vintage data (Croushore; Faust–Wright), FRED macro data, and recent applied-DML replication studies (Knaus; Strittmatter; Baiardi–Naghi).
