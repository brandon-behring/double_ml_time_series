# Implementations and software tooling

DML and causal-inference software: DoubleML (R/Python), EconML (Microsoft), DoWhy (PyWhy), CausalML (Uber), hdm (R), grf (R, generalized random forests), `tmle3` / `causalweight` / `causaltune` / `xtdml` / `ddml` (Stata), and CausalImpact (Bayesian-STS). Tools, not papers — these are the implementing toolchain.
