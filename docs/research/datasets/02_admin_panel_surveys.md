# Administrative / Panel Surveys

Long-running individual / household panels — PSID, NLSY79/97, SIPP, MEPS, CPS, ACS, HRS, NHIS, NHANES (US); EU-SILC, EU-LFS, UKHLS, GSOEP (EU/UK); OECD and World Bank cross-country panels. These are the canonical microdata sources for panel-DML applications in labor, health, and policy.

---

## B1. American Community Survey (ACS) Public Use Microdata Sample (PUMS)

- **American Community Survey (ACS) Public Use Microdata Sample (PUMS)** — US Government open data (2005).
  - **Source:** https://www.census.gov/programs-surveys/acs/
  - **Access:** direct download; auth_required: N
  - **Schema:** rows: approximately 3 million per year (5-year file ~15 million), task_family: tabular
  - **Size+License:** license: public domain (U.S. federal government work)
  - **Tasks:** U.S. Census Bureau. American Community Survey (ACS), 2005-present. Public Use Microdata Sample (PUMS).
  - **Status:** Verified.
