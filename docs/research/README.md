- Skim `papers/README.md` for the lookup recipes to find canonical references by topic.
- Use `bibliography_overlap.md` to see which book-bib entries are already covered (23/34) vs. which were seeded into pass-2 (4) vs. which are out-of-scope textbooks (7).

**If you're a future Claude agent grounding in DML:**

- Cite specific entries by file + bullet position (e.g., "see `papers/04_estimator_temporal.md` § entry for Lewis & Syrgkanis 2020 NeurIPS").

## Scope boundaries

This dossier covers DML applied to **time-series and panel data**. Deliberately excluded:

- Synthetic control / synthetic DiD (Abadie et al.) — adjacent but methodologically distinct.
- DiD with staggered adoption (Callaway-SantAnna, de Chaisemartin-d'Haultfœuille, Borusyak et al.) — separate plan.
- Pure time-series forecasting (ARIMA / state-space / TS-transformer) — different evaluation conventions.
- Causal discovery / DAG learning — different end-goal.
- Mediation analysis under DML — small adjacent literature.
- Off-policy evaluation / contextual bandits — different evaluation (regret-based).
- LLM-based causal reasoning — agent-side rather than estimator-side.
- Survival / duration outcomes under DML — specialized literature.

See `research_plan.md` § Out-of-scope for the complete list with rationale.
