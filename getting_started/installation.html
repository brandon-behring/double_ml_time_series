Installation#

Requirements#

  • Python 3.11+

  • NumPy, SciPy, pandas, scikit-learn, statsmodels

  • Optional: FRED API key for live macroeconomic data

  • Optional docs dependencies for Sphinx builds

Install From Source#

cd double_ml_time_series
venv/bin/python -m pip install -e ".[dev]"

For documentation work:

venv/bin/python -m pip install -e ".[dev,docs]"

Verify Installation#

venv/bin/python -c "from dml_ts.dml import double_ml, TemporalPLRDML, RollingWindowDML; print('OK')"
venv/bin/python -c "from dml_ts.data import FREDLoader, create_synthetic_fred_data; print('OK')"
venv/bin/python -c "from dml_ts.validation import create_insurance_dgp; print('OK')"

Optional: FRED API Key#

Live FRED access requires an API key:

export FRED_API_KEY="your_key_here"

Use create_synthetic_fred_data() for offline examples and tests that should not depend on network access.

Building Documentation#

venv/bin/python -m sphinx -b html -W --keep-going docs/sphinx docs/sphinx/_build/html
