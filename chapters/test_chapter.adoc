== Code Block Test

Here's a simple Python example demonstrating a Double ML estimation:

[source,python]
----
import numpy as np
from econml.dml import LinearDML
from sklearn.linear_model import LassoCV
from sklearn.ensemble import RandomForestRegressor

# Generate synthetic data
np.random.seed(42)
n = 1000
X = np.random.randn(n, 5)  # Confounders
T = X[:, 0] + np.random.randn(n)  # Treatment
Y = 2 * T + X[:, 1] + np.random.randn(n)  # Outcome with true effect = 2

# Fit DML model
dml = LinearDML(
    model_y=RandomForestRegressor(n_estimators=100, n_jobs=48),
    model_t=LassoCV(cv=5),
    random_state=42
)
dml.fit(Y, T, X=X)

# Estimated treatment effect
effect = dml.effect(X)
print(f"Average Treatment Effect: {effect.mean():.3f}")
----

The code above demonstrates the basic DML workflow with hardware optimization (`n_jobs=48` for our 64-core Threadripper).

== Mathematical Equations Test

The Frisch-Waugh-Lovell (FWL) theorem states that for the regression:

[stem]
++++
Y = \beta_0 + \beta_1 T + \beta_2 X + \epsilon
++++

The coefficient stem:[\beta_1] can be obtained by:

1. Regress stem:[Y] on stem:[X] to get residuals stem:[\tilde{Y}]
2. Regress stem:[T] on stem:[X] to get residuals stem:[\tilde{T}]
3. Regress stem:[\tilde{Y}] on stem:[\tilde{T}]

Formally:

[stem]
++++
\hat{\beta}_1 = \frac{\text{Cov}(\tilde{Y}, \tilde{T})}{\text{Var}(\tilde{T})}
++++

This is the foundation of Double Machine Learning, as discussed in <<neyman-orthogonality>>.
